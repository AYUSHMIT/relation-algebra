Code

Formal proofs and libraries

  • Relation Algebra and KAT in Coq (includes a tactic for deciding Kleene algebras with Tests).
  • Certification of robust reconfiguration algorithms (joint work with Fabienne Boyer and Olivier Gruber).
  • Algebraic Tools for Binary Relations (includes a tactic for deciding Kleene algebras---joint work with Thomas Braibant).
  • Plugin for rewriting modulo associativity and commutativity (joint work with Thomas Braibant)).
  • Formalisation of the entire ICALP'05 article.
  • Certified compiler from recursive functions to Minski machines.

Algorithms

  • Symbolic Algorithms for Language Equivalence and Kleene Algebra with Tests (KAT).
  • A new algorithm for NFA language equivalence.
  • Checking must testing using bisimulations up to congruence.
  • Optimised proof search algorithm for cyclic linear logic.

Abstract Machines

  • Distributed implementation of an abstract machine for Safe Ambient.
  • Graphical machine for interpreting various notions of Mobile Ambients in a funny way.
